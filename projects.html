My Projects

Welcome to my projects page. I have listed these projects in reverse chronological order.
If you want a deeper look at the code, take a look at my GitHub repository.

My Portfolio Website: GitHub

The portfolio website started as a project where we were introduced to html and css one morning and had two days to create a portfolio website. This site will continue to improve as I become more familiar with HTML and CSS techniques.

Linq Problems: GitHub

This project demonstrates my ability to query objects in C# using LINQ(Language Integrated Query).

Humane Society: GitHub

This project demonstrates my ability to create a SQL database, seed a SQL database, perform CRUD database functions (Create, Remove, Update, and Delete), as well as create specific queries for desired data.

Sweepstakes Manager: GitHub

The Sweepstakes project simulated a Sweepstakes Manager program that a Marketing Firm would use to manage the sweepstakes of their clientele. This program utilized a user interface whenever application needed user input, as well as a Stack and Queue data structure depending on user choice. Other concepts used include dependency injection, factory design pattern, and observer design pattern.

Binary Search Tree: GitHub

In the Binary Search Tree project, I created a data structure that stores items in memory and is easily sortable and searchable.

Custom List: GitHub

In the Custom List Project, I created my own unique data structure. This was a Custom List that was an Array with a customized wrapper that allowed it to have list-like capabilities. We were not allowed to use built in List or Array methods or properties. I developed override methods for ToString and .Zip methods as well as overloaded the + and - operators to add and subtract my custom data structures to and from each other.

Lemonade Stand: GitHub

The Lemonade Stand Project is a business simulation game where a player is able to buy supplies for a lemonade stand and then the sales are determined by the palatability of your lemonade as well as weather.

Rock, Paper, Scissors, Lizard, Spock: GitHub

A new take on an old classic, the RPSLS game simulates Rock, Paper, Scissors with two added twists: Lizard beats Spock and Paper and Spock beats Rock and Scissors.

Robots vs. Dinosaurs: GitHub

A simulation game of three robots vs three dinosaurs.
